# Convert SCSS to CSS
Author: [Ivan Piedra](https://twitter.com/ivapie)

### Installation

Install the dependencies and devDependencies

```sh
$ npm install
$ npm run convert
```

The 'npm run convert' works for:

- Convert ./components/**/*.scss to ./build/styles.css
- Minify .css
- Clear .css
- Check for duplicates classes

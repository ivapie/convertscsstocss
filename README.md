# Convert SCSS to CSS
Author: [Ivan Piedra](https://twitter.com/ivapie)

### Installation

Install the dependencies and devDependencies and run the convert command for

```sh
$ npm install
$ npm run convert
```

Run convert command for:

- Convert .scss to css
- Concat .css
- Minify .css
- Clear .css
- Check for duplicates classes
- Save a .css (styles.css)
